# AGF211_2026
This repository contains the measurement data and code from the Mooring group of AGF211 2026. The data consist of SeaGuard measurements, temperature logger data and  SeaBird measurements.  

**Datasets**:  
Seaguard_1.txt: SeaGuard 1 at 5.5m depth  
5650-2803-0-2026-03-18T13-20-00.040Z.csv : SeaGuard 2 at 14.5 m depth  
sbe37smp-rs232_03727873_2026_03_21.cnv: Seabird 1 at 2 m depth 
Seabird5453.asc : Seabird 2 at 8.5 m depth  
sbe37smp-rs232_03727626_2026_03_21.cnv: Seabird 3 at 12 m depth  
SBE37SM-RS232_03710963_2026_03_21.cnv: Seabird 4 at 18 m depth  
(depth/pressure sensor of Seabird 4 broken)  

**Code files**
Code_currents_tides_salinity.ipynb: Code for the current and tidal analysis, salinity time-series plot, tidal constituents
